# Assets & Images Directory

This directory contains visual assets (screenshots, images, icons) for the Auto Bidder project.

## 📁 Current Files

- `dashboard.png` - Main dashboard screenshot
- `analytics.png` - Analytics and metrics dashboard screenshot

## 📸 Adding New Screenshots

When adding new screenshots to the project:

1. **Take high-quality screenshots** (1200-1600px width recommended)
2. **Optimize file size** (aim for < 500KB per image using [TinyPNG](https://tinypng.com/))
3. **Use descriptive names** (e.g., `proposal-builder.png`, `knowledge-base.png`)
4. **Save to this directory** (`assets/images/`)
5. **Reference in README.md**:
   ```markdown
   ![Description](./assets/images/your-screenshot.png)
   ```

## 🎨 Image Guidelines

- **Format**: PNG for UI screenshots, JPG for photos
- **Dimensions**: 1200-1600px width for screenshots
- **File size**: < 500KB (optimized)
- **Naming**: Use kebab-case (e.g., `feature-name-view.png`)
- **Privacy**: Remove any sensitive API keys or personal information

## 🛠️ Optimization Tools

- [TinyPNG](https://tinypng.com/) - Online PNG/JPG compression
- [ImageOptim](https://imageoptim.com/) - Mac app for image optimization
- [Squoosh](https://squoosh.app/) - Google's image compression tool
